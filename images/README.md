# 图片资源说明

这个文件夹用于存放网站所需的图片资源。

## 需要添加的图片

请将以下图片文件放入此文件夹：

### 1. profile.jpg
- **用途**：个人照片（关于我部分）
- **建议尺寸**：400x400 像素或更大
- **格式**：JPG 或 PNG
- **说明**：你的个人照片，最好是正方形

### 2. project1.jpg
- **用途**：项目1的预览图
- **建议尺寸**：800x600 像素
- **格式**：JPG 或 PNG
- **说明**：项目的截图或预览图

### 3. project2.jpg
- **用途**：项目2的预览图
- **建议尺寸**：800x600 像素
- **格式**：JPG 或 PNG

### 4. project3.jpg
- **用途**：项目3的预览图
- **建议尺寸**：800x600 像素
- **格式**：JPG 或 PNG

## 如何获取占位图片

如果你暂时没有图片，可以使用以下占位图服务：

1. **Unsplash Source**
   - 个人照片：`https://source.unsplash.com/400x400/?portrait`
   - 项目图片：`https://source.unsplash.com/800x600/?technology`

2. **Placeholder.com**
   - `https://via.placeholder.com/400x400`
   - `https://via.placeholder.com/800x600`

3. **Lorem Picsum**
   - `https://picsum.photos/400/400`
   - `https://picsum.photos/800/600`

## 优化建议

- 使用工具压缩图片以提高加载速度（推荐：TinyPNG）
- 确保图片清晰但不要过大（单张图片建议不超过500KB）
- 使用 WebP 格式可以获得更好的压缩效果
